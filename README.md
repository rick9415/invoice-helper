# 銷售額與稅額快速計算器

這是一個針對連續輸入發票資訊設計的快速計算工具。

## 功能特點
- **流暢輸入**：支援 Enter 鍵快速提交，自動切換銷售額與稅額輸入。
- **即時統計**：自動計算總銷售額、總稅額及含稅總計。
- **響應式設計**：完美支援手機與桌面瀏覽器。
- **隱私安全**：純前端應用，資料不會上傳至伺服器。

## 如何使用
1. 輸入第一張發票的**銷售額**，按下 Enter 或 "+" 按鈕。
2. 輸入該發票的**稅額**，按下 Enter 或 "+" 按鈕。
3. 重複上述步驟即可連續累計。
4. 如有輸入錯誤，點擊右側的刪除圖示即可移除該筆記錄。

## 部署說明
本專案支援 GitHub Pages 部署：
1. 在 GitHub 上建立一個新的 Repository。
2. 將此專案推送至 GitHub。
3. 進入 Repository 的 **Settings > Pages**。
4. 在 **Build and deployment** 下選擇 `main` 分支並儲存。
5. 稍待片刻即可透過 GitHub 提供的網址訪問。
